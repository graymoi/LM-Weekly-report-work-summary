## 二、组织架构

### 2.1 内设机构（8个职能处室）

| 序号 | 机构名称 | 主要职责 |
|------|---------|---------|
| 1 | **办公室** | 负责委机关内部政务和内部管理工作。承担公文处理、文书档案、机要保密、会务办理、信息报送、政务公开、工作督查、绩效考核、建议提案办理、应急值班、政务网站建设、后勤保障工作。牵头负责安全生产、信访维稳工作并协调处置应急突发事件。负责本系统财务管理、监督、审计工作。 |
| 2 | **党建工作室** | 承担委党委日常工作。承担委系统党组织建设、作风建设、党风廉政建设和党员队伍建设工作。负责委系统宣传和精神文明建设工作。协调指导机关和所属单位工会、共青团、妇联等群团组织工作；承担政工职称评定工作。负责系统机构编制、人事管理、教育培训、任用调配、录用考核和监督管理工作。负责工资、福利、养老保险等工作。指导事业单位工作人员公开招聘、考核聘用和岗位设置管理工作。组织行业职工专业技术（技能）培训、继续教育、职称评审、考工定级等工作。负责系统人才队伍建设。负责干部出国（境）管理和干部档案管理工作。负责离退休干部的管理和服务工作。 |
| 3 | **园林建设管理室** | 负责拟订城市园林绿化专项规划和城市园林绿化设施年度建设计划，并组织实施。负责城市园林绿化和城市公园行业管理工作。负责拟订城市园林绿化建设工程及养护相关质量标准、规范，并监督执行。承担区绿化委员会办公室日常工作。负责组织指导全区义务植树、城乡绿化、古树名木保护工作。承担园林绿化施工企业失信信息管理工作。对全区重大活动中涉及城市园林绿化工作及城区重大疫情的防治实施统一指挥。指导公园、动物园和风景游览区园林绿化工作。负责指导各类公园重大节日、重要活动期间游园预案制定、绿化景点布置工作。负责城市园林绿化宣传教育、科技项目研究、成果推广应用及对外合作交流。负责组织推动城市园林绿化科技项目的鉴定和评估。负责协调全区各部门绿化工作。组织全区绿化先进评比表彰工作。 |
| 4 | **环境卫生管理室** | 负责拟订环卫作业标准、规范和定额，并监督执行。负责指导、监督和检查公共场所、道路、公厕、居民社区等区域的环卫工作。负责制定环卫行业市场准入制度并组织实施。承担区清雪指挥部办公室日常工作。拟订全区机动车辆清洗行业的相关规范和行业标准，对机动车辆清洗管理工作进行指导和监督检查。负责全区生活废弃物清扫、收集、运输和处置处理的统一监督管理。负责推动协调落实生活垃圾分类管理工作。负责渣土治理工作。推动生活废弃物再生利用和无害化处理工作。负责组织重大会议活动、重要节假日环境秩序保障工作。负责环境卫生设施的使用和管理。负责环境卫生作业管理和综合统计工作。负责组织推动环境卫生科技项目的鉴定和评估。 |
| 5 | **公共事业管理室** | 组织编制燃气、供热公用基础设施的发展规划和专项规划，参与拟订年度建设计划。负责既有燃气、供热等公用基础设施特许经营项目监督管理，参与新建燃气、供热等公用基础设施特许经营项目监督管理。 |
| 6 | **市政道路桥梁管理室** | 组织编制城市道路桥梁等市政基础设施的发展规划和专项规划，参与拟订年度建设计划。负责既有城市道路桥梁等市政基础设施特许经营项目监督管理。负责夜景灯光、景观照明、功能性照明管理工作。负责城市道路管线井监督管理。负责临时占用城市道路监督管理。负责城市停车监督管理。负责本单位防汛工作。参与新建城市道路桥梁等市政基础设施特许经营项目监督管理。 |
| 7 | **规划建设计划室** | 组织编制市容市貌、环境卫生、城乡绿化以及燃气、供热、城市道路桥梁等市政公用基础设施事业发展规划、专项规划。组织拟订市容市貌、环境卫生、城乡绿化等市政公用基础设施年度建设计划。参与编制燃气、供热、城市道路桥梁等市政公用基础设施年度建设计划。负责审查系统内基本建设改造项目方案。负责建设项目的规划选址和可行性研究。协调推进城市管理重点项目建设。配合做好市容市貌、环境卫生、城乡绿化、市政公用基础设施用地管理工作。组织指导建设、代建单位施工管理。负责景观照明、市容景观的建设工作。负责城市管理总体统计工作，地下管线（天津市地下市政基础设施普查和综合管理信息平台）的统计工作。监督、协调城市管理的数字化、智慧化建设与运行工作。组织开展城市管理发展战略、重大问题、重点课题、政策执行情况的调查研究、鉴定评估验收、科技成果应用推广及对外技术交流与合作。 |
| 8 | **城市管理考核室** | 负责拟订完善城市管理考核指标体系和管理制度，并组织实施。组织对区级有关部门和区城市管理工作进行监督考核。负责户外广告设置、城市家具及建筑外檐等城市景观的管理及安全监督工作。牵头组织协调环境综合整治工作并组织实施。负责城乡结合部地区的市容环境秩序的综合整治、协调和督办工作。牵头组织重大节日、重要活动期间的市容环境布置和环境秩序保障工作。负责行政执法、行政复议、行政诉讼和行业普法宣传工作。拟订城市管理权责清单。负责编制全区城市管理综合行政执法工作规划，制定工作目标、标准，并组织实施和考核。组织、指挥、调度全区城市管理综合行政执法机构开展执法活动。负责重大和跨区域违法违规案件的查处及执法协调工作。负责城市管理领域安全生产监管，会同相关部门协调处置应急突发事件。负责城市管理综合行政执法监督工作。负责城市管理综合执法队伍的考核、监察工作。负责全区城市管理综合行政执法人员的培训、考核和执法资格审查工作。负责城市管理综合行政执法信访工作。 |

## 三、主要职责

### 3.1 核心职能

| 职能类别 | 具体内容 |
|---------|---------|
| **市容市貌管理** | 贯彻执行国家有关城市管理的法律、法规和方针、政策、标准，落实有关地方性法规、政府规章，拟订有关政策、标准，并组织实施 |
| **环境卫生管理** | 负责环境卫生的组织管理和监督检查工作，负责全市生活废弃物清扫、收集、运输和处置的统一监督管理 |
| **园林绿化管理** | 负责城市建成区园林绿化工作，负责公园管理 |
| **市政基础设施管理** | 组织编制有关市容市貌、环境卫生、城市园林绿化以及燃气、供热、城市道路桥梁等市政公用基础设施专项规划 |
| **城市管理考核** | 拟订完善城市管理考核指标体系和管理制度，并组织实施 |
| **行政执法** | 负责城市管理综合行政执法工作，查处违法违规案件 |

### 3.2 重点工作

1. **市容市貌建设项目**：组织实施市容市貌建设项目
2. **生活废弃物管理**：负责全市生活废弃物清扫、收集、运输和处置的统一监督管理
3. **城市园林绿化**：负责城市建成区园林绿化工作
4. **公园管理**：负责公园管理
5. **城市管理考核**：组织对区级有关部门和区城市管理工作进行监督考核
6. **环境综合整治**：牵头组织协调环境综合整治工作并组织实施
