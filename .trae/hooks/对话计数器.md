# 对话计数器 Hook

## 概述

每次对话后自动更新对话计数器，达到10轮时触发对话日志存档。

---

## 触发条件

- 每次对话后自动触发

---

## 工作流程

```
对话开始
  ↓
对话计数
  ↓
  ├── 读取计数器
  ├── 更新计数器
  └── 检查是否达到10轮
  ↓
达到10轮？
  ↓
  ├── 是 → 调用对话日志存档 Skill
  │       ↓
  │   创建存档文件
  │       ↓
  │   提取知识精华
  │       ↓
  │   明确提示用户
  │       ↓
  │   重置计数器
  │
  └── 否 → 继续对话
          ↓
      等待下一轮对话
```

## 计数器管理

### 计数器位置

```
.trae\data\对话计数器.json
```

### 计数器格式

```json
{
  "count": 0,
  "last_update": "2026-03-07T23:12:00",
  "last_archive": "2026-03-07T23:09:00",
  "archive_count": 1
}
```

### 计数器操作

```powershell
# 读取计数器
Get-Content ".trae\data\对话计数器.json" | ConvertFrom-Json

# 更新计数器
$count = Get-Content ".trae\data\对话计数器.json" | ConvertFrom-Json
$count.count += 1
$count.last_update = Get-Date -Format "yyyy-MM-ddTHH:mm:ss"
$count | ConvertTo-Json | Set-Content ".trae\data\对话计数器.json"

# 重置计数器
$count = Get-Content ".trae\data\对话计数器.json" | ConvertFrom-Json
$count.count = 0
$count.last_archive = Get-Date -Format "yyyy-MM-ddTHH:mm:ss"
$count.archive_count += 1
$count | ConvertTo-Json | Set-Content ".trae\data\对话计数器.json"
```

## 触发逻辑

```powershell
# 每次对话后执行
$countPath = ".trae\data\对话计数器.json"

# 检查计数器文件是否存在
if (-not (Test-Path $countPath)) {
    # 创建计数器文件
    New-Item -ItemType File -Path $countPath -Force | Out-Null
    $count = @{
        count = 0
        last_update = Get-Date -Format "yyyy-MM-ddTHH:mm:ss"
        last_archive = Get-Date -Format "yyyy-MM-ddTHH:mm:ss"
        archive_count = 0
    }
    $count | ConvertTo-Json | Set-Content $countPath
} else {
    # 读取计数器
    $count = Get-Content $countPath | ConvertFrom-Json
}

# 更新计数器
$count.count += 1
$count.last_update = Get-Date -Format "yyyy-MM-ddTHH:mm:ss"

# 检查是否达到10轮
if ($count.count -ge 10) {
    # 触发对话日志存档
    Write-Host "📝 对话日志存档触发：已达到10轮对话" -ForegroundColor Cyan

    # 调用对话日志存档 Skill
    # ... 调用逻辑 ...

    # 重置计数器
    $count.count = 0
    $count.last_archive = Get-Date -Format "yyyy-MM-ddTHH:mm:ss"
    $count.archive_count += 1

    Write-Host "✅ 计数器已重置" -ForegroundColor Green
    Write-Host "📊 累计存档次数: $($count.archive_count)" -ForegroundColor Yellow
}

# 保存计数器
$count | ConvertTo-Json | Set-Content $countPath

Write-Host "🔄 对话计数已更新: $($count.count)/10" -ForegroundColor Gray
```

## 集成方式

### 方式1：手动触发

**适用场景**：每次对话后手动执行

```powershell

# 触发: 每次 `git commit` 成功后
```

## 常见问题

### 问题1：计数器文件不存在

**症状**: 执行时提示找不到计数器文件

**原因**: 计数器文件没有初始化

**解决方案**:
```powershell
# 检查计数器文件是否存在
Test-Path ".trae\data\对话计数器.json"

# 创建计数器文件
New-Item -ItemType File -Path ".trae\data\对话计数器.json" -Force

# 初始化计数器
@{
    count = 0
    last_update = Get-Date -Format "yyyy-MM-ddTHH:mm:ss"
    last_archive = Get-Date -Format "yyyy-MM-ddTHH:mm:ss"
    archive_count = 0
} | ConvertTo-Json | Set-Content ".trae\data\对话计数器.json"
```

### 问题2：计数器不更新

**症状**: 执行后计数器没有变化

**原因**: 权限问题或文件锁定

**解决方案**:
```powershell
# 检查文件权限
icacls ".trae\data\对话计数器.json"

# 设置文件权限
icacls ".trae\data\对话计数器.json" /grant Everyone:F
```

### 问题3：存档不触发

**症状**: 达到10轮后没有触发存档

**原因**: 对话日志存档 Skill 没有正确调用

**解决方案**:
```powershell
# 检查对话日志存档 Skill 是否存在
Test-Path ".trae\skills\对话日志存档\SKILL.md"
```
