# 对话日志存档 Skill

## 概述

每10轮对话自动存档，萃取知识精华，封装可复用内容，实现AI自我进化。

---

## 触发条件

- **自动触发**: 达到10轮对话（由 personal_rules.md 中的对话计数规则触发）
- **手动触发**: 用户明确要求存档

## 执行流程

```
达到10轮对话
    │
    ├── 1. 创建存档文件
    │       └── 输出成果\对话日志\YYYY-MM-DD\对话日志_YYYY-MM-DD.md
    │
    ├── 2. 记录完整对话
    │       └── 逐轮记录用户和AI的对话
    │
    ├── 3. 萃取知识精华
    │       ├── 核心洞察：问题本质、反直觉发现、跨领域连接
    │       ├── 经验总结：可复用方法、成功做法、失败教训
    │       ├── 待解决问题：未完成任务、遗留问题
    │       └── 后续行动：具体步骤、优先级、时间节点
    │
    ├── 4. 提取零件素材
    │       ├── 可复用流程：触发条件 + 执行步骤
    │       ├── 可复用模式：适用场景 + 处理逻辑
    │       └── 可复用代码：用途 + 代码片段
    │
    ├── 5. 自动封装
    │       ├── 识别可复用流程 → 生成 Skill
    │       ├── 识别重复模式 → 生成 Rule
    │       └── 识别触发点 → 生成 Hook
    │
    ├── 6. 重置计数器
    │       ├── count = 0
    │       ├── last_archive = 当前时间
    │       └── archive_count + 1
    │
    └── 7. 明确提示用户
            └── 显示存档信息和查看链接
```

## 知识萃取规则

### 核心洞察
- 对问题本质的深刻理解
- 反直觉的发现
- 跨领域的连接
- 新的认知框架

### 经验总结
- 可复用的方法论
- 成功的做法
- 失败的教训
- 最佳实践

### 待解决问题
- 未完成的任务
- 需要进一步探索的问题
- 遗留问题
- 技术债务

### 后续行动
- 具体的行动步骤
- 优先级排序
- 时间节点
- 负责人

## 零件素材提取规则

### 可复用流程
**识别条件**：
- 出现3次以上的操作序列
- 标准化的工作流程
- 明确的输入输出

**提取格式**：
```
[流程名称]
触发条件: [什么情况下执行]
执行步骤:
  1. [步骤1]
  2. [步骤2]
  3. [步骤3]
输出: [结果]
```

### 可复用模式
**识别条件**：
- 重复出现的决策逻辑
- 标准化的判断规则
- 固定的行为模式

**提取格式**：
```
[模式名称]
适用场景: [什么情况下使用]
处理逻辑:
  IF [条件1] THEN [处理1]
  IF [条件2] THEN [处理2]
  DEFAULT [默认处理]
```

### 可复用代码
**识别条件**：
- 可复用的代码片段
- 通用的工具函数
- 标准化的配置

**提取格式**：
```
[代码名称]
用途: [解决什么问题]
代码:
  [代码片段]
```

## 自动封装规则

### 生成 Skill
**触发条件**：
- 发现可复用的流程
- 用户明确要求封装

**生成位置**：`.trae\skills\[名称]\SKILL.md`

**生成格式**：
```markdown
# [名称] Skill

## 概述
[功能描述]

## 触发条件
- [触发条件]

## 执行流程
[流程步骤]
```

### 生成 Rule
**触发条件**：
- 发现重复的模式
- 用户明确要求添加规则

**生成位置**：`.trae\rules\[名称].md`

**生成格式**：
```markdown
# [名称] Rule

## 适用场景
- [场景]

## 规则内容
[规则描述]
```

### 生成 Hook
**触发条件**：
- 发现自动触发点
- 用户明确要求自动化

**生成位置**：`.trae\hooks\[名称].md`

**生成格式**：
```markdown
# [名称] Hook

## 触发时机
- [触发条件]

## 执行逻辑
[逻辑描述]
```

## 计数器管理

### 文件位置
```
.trae\data\对话计数器.json
```

### 数据格式
```json
{
  "count": 0,
  "last_update": "2026-03-15T12:00:00",
  "last_archive": "2026-03-15T12:00:00",
  "archive_count": 1
}
```

### 操作规则
- **更新**: 每次对话后 count + 1
- **重置**: 存档后 count = 0
- **记录**: 更新 last_update 和 last_archive
