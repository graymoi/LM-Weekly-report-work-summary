# 对话日志存档 Skill

## 概述

实现对话日志存档的具体逻辑，包括创建存档文件、提取知识精华、更新知识库、建立问题-答案库、自动封装可复用内容。

**注意**: 对话计数逻辑已移至 `.trae\hooks\对话计数器.md`，本 Skill 只负责存档执行。

---

## 核心功能

### 1. 创建存档文件
- 创建存档目录
- 创建存档文件
- 写入存档内容

### 2. 提取知识精华
- 提取核心洞察
- 提取经验总结
- 提取待解决问题
- 提取后续行动

### 3. 更新知识库
- 更新项目库（项目跟踪记录、人物库、案例库、政策库）
- 更新个人知识库（技能清单、经验库、学习记录、能力图谱）

### 4. 建立问题-答案库
- 提取咨询过的问题
- 提取对应的答案
- 建立问题-答案索引
- 支持快速检索

### 5. 自动封装可复用内容
- 识别可复用的流程 → 自动生成 Skill
- 识别重复的模式 → 自动生成 Rule
- 识别自动触发点 → 自动生成 Hook

### 6. 明确提示用户
- 显示存档信息
- 提供查看链接
- 显示知识库更新情况
- 显示自动封装内容

---

## 触发条件

### 自动触发

- **达到10轮对话**: 由对话计数器 Hook 触发
- **每日结束**: 由对话计数器 Hook 触发
- **手动触发**: 用户手动调用

## 自动化配置

### 任务处理 Skill 集成

**配置**: 在任务处理 Skill 中，每次对话后自动更新计数器

**触发**: 每次对话后

**检查**: 达到10轮时自动触发存档

### Git Hooks 集成

#### Post-commit Hook

**功能**: 在 Git 提交后自动存档对话日志

**触发时机**: 每次 `git commit` 成功后

**配置文件**: `.git/hooks/post-commit`

**注意**: 对话计数和触发逻辑由 `.trae\hooks\对话计数器.md` 处理，本 Hook 仅用于触发存档

```powershell
# Git post-commit hook - 对话日志存档
# 功能: 在 Git 提交后自动存档对话日志
# 触发: 每次 `git commit` 成功后
# 生成: 对话日志存档文件

# 调用对话计数器 Hook
& ".trae\hooks\对话计数器.md"

exit 0
```

## 常见问题

### 问题1：计数器不更新

**症状**: 对话轮数没有增加

**原因**: 计数器文件不存在或权限问题

**解决方案**:
```powershell
# 检查计数器文件是否存在
Test-Path ".trae\data\对话计数器.json"

# 创建计数器文件
New-Item -ItemType File -Path ".trae\data\对话计数器.json" -Force

# 初始化计数器
@{
    count = 0
    last_update = Get-Date -Format "yyyy-MM-ddTHH:mm:ss"
    last_archive = Get-Date -Format "yyyy-MM-ddTHH:mm:ss"
    archive_count = 0
} | ConvertTo-Json | Set-Content ".trae\data\对话计数器.json"
```

### 问题2：存档文件不生成

**症状**: 达到10轮对话后没有生成存档文件

**原因**: 存档逻辑没有执行或权限问题

**解决方案**:
```powershell
# 检查存档目录是否存在
Test-Path "输出成果\对话日志\YYYY-MM-DD"

# 创建存档目录
New-Item -ItemType Directory -Path "输出成果\对话日志\YYYY-MM-DD" -Force

# 手动触发存档
# ... 存档逻辑 ...
```

### 问题3：看不到明确提示

**症状**: 存档完成但没有看到提示

**原因**: 提示信息没有显示或被忽略

**解决方案**:
- 确保提示信息清晰可见
- 使用醒目的格式（如表格、图标）
- 提供查看链接
