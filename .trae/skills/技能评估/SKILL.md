# 技能评估 Skill

## 概述

评估现有技能的效果、性能和适用性，提供量化指标和优化建议。

---

## 核心功能

### 1. 使用统计
- 记录使用频率
- 统计触发次数
- 计算使用时长
- 分析使用模式

### 2. 效果评估
- 评估输出质量
- 测量效率提升
- 分析用户满意度
- 计算投资回报

### 3. 性能分析
- 响应时间分析
- 资源使用监控
- 错误率统计
- 稳定性评估

### 4. 优化建议
- 识别优化点
- 生成优化方案
- 评估优化效果
- 提供实施建议

---

## 触发条件

### 显式触发

- "评估技能"、"技能评估"、"评估效果"
- "技能性能"、"技能效果"、"技能分析"
- "技能报告"、"评估报告"

### 自动触发

- 定期评估（每周）
- 技能使用后
- 检测到性能问题
- 用户反馈后

## 评估指标

### 使用指标

| 指标 | 说明 | 计算方式 |
|------|------|---------|
| 使用次数 | 总使用次数 | COUNT(使用记录) |
| 日均使用次数 | 每日平均使用次数 | 使用次数 / 天数 |
| 触发成功率 | 成功触发比例 | 成功次数 / 总次数 |
| 使用时长 | 平均每次使用时长 | SUM(时长) / 使用次数 |

### 效果指标

| 指标 | 说明 | 计算方式 |
|------|------|---------|
| 准确性 | 输出准确性评分 | 用户评分 / 5.0 |
| 完整性 | 输出完整性评分 | 用户评分 / 5.0 |
| 效率提升 | 效率提升百分比 | (节省时间 / 原时间) × 100% |
| 用户满意度 | 用户满意度评分 | 用户评分 / 5.0 |

### 性能指标

| 指标 | 说明 | 计算方式 |
|------|------|---------|
| 响应时间 | 平均响应时间 | SUM(响应时间) / 次数 |
| P95 响应时间 | 95% 请求响应时间 | PERCENTILE(响应时间, 95) |
| 成功率 | 执行成功率 | 成功次数 / 总次数 |
| 资源使用 | 平均资源使用 | AVG(资源使用) |

---

## 评分标准

### 综合评分

```
综合评分 = (使用评分 × 0.3 + 效果评分 × 0.4 + 性能评分 × 0.3)
```

### 分项评分

| 评分范围 | 等级 | 说明 |
|---------|------|------|
| 9.0 - 10.0 | 优秀 | 表现非常好 |
| 7.5 - 8.9 | 良好 | 表现良好 |
| 6.0 - 7.4 | 一般 | 表现一般 |
| 4.0 - 5.9 | 较差 | 需要改进 |
| 0.0 - 3.9 | 很差 | 需要重大改进 |

## 最佳实践

### 1. 定期评估
- 每周自动评估一次
- 评估周期保持一致
- 对比历史数据

### 2. 多维度评估
- 使用、效果、性能多维度
- 量化指标 + 质化评估
- 用户反馈 + 系统数据

### 3. 持续优化
- 基于评估结果优化
- 追踪优化效果
- 持续改进

### 4. 数据驱动
- 基于真实数据评估
- 避免主观判断
- 量化评估结果
