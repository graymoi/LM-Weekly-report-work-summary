# 性能监控 Skill

## 概述

实时监控系统性能，包括技能执行、资源使用、响应时间等，提供性能预警和优化建议。

---

## 核心功能

### 1. 实时监控
- 监控技能执行
- 跟踪资源使用
- 测量响应时间
- 记录错误日志

### 2. 性能分析
- 分析性能趋势
- 识别性能瓶颈
- 评估系统负载
- 预测性能变化

### 3. 预警通知
- 性能下降预警
- 资源超限预警
- 错误率升高预警
- 系统异常预警

### 4. 优化建议
- 识别优化点
- 生成优化方案
- 评估优化效果
- 提供实施建议

## 预警规则

### CPU 使用率预警

```python
def check_cpu_usage(cpu_usage):
    if cpu_usage > 90:
        return {
            'level': 'critical',
            'message': f'CPU 使用率过高: {cpu_usage}%'
        }
    elif cpu_usage > 80:
        return {
            'level': 'warning',
            'message': f'CPU 使用率偏高: {cpu_usage}%'
        }
    else:
        return {
            'level': 'normal',
            'message': f'CPU 使用率正常: {cpu_usage}%'
        }
```

### 内存使用预警

```python
def check_memory_usage(memory_usage, total_memory):
    usage_percent = (memory_usage / total_memory) * 100
    
    if usage_percent > 90:
        return {
            'level': 'critical',
            'message': f'内存使用率过高: {usage_percent:.1f}%'
        }
    elif usage_percent > 85:
        return {
            'level': 'warning',
            'message': f'内存使用率偏高: {usage_percent:.1f}%'
        }
    else:
        return {
            'level': 'normal',
            'message': f'内存使用率正常: {usage_percent:.1f}%'
        }
```

### 响应时间预警

```python
def check_response_time(response_time):
    if response_time > 10:
        return {
            'level': 'critical',
            'message': f'响应时间过长: {response_time:.2f} 秒'
        }
    elif response_time > 5:
        return {
            'level': 'warning',
            'message': f'响应时间偏长: {response_time:.2f} 秒'
        }
    else:
        return {
            'level': 'normal',
            'message': f'响应时间正常: {response_time:.2f} 秒'
        }
```
