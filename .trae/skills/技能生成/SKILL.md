# 技能生成 Skill

## 概述

基于用户需求和最佳实践，自动生成新的技能，实现技能的快速创建和部署。

---

## 核心功能

### 1. 需求分析
- 分析用户需求
- 识别技能类型
- 确定功能范围
- 评估可行性

### 2. 技能设计
- 设计技能架构
- 定义触发条件
- 规划工作流程
- 设计输出格式

### 3. 代码生成
- 生成技能代码
- 实现核心功能
- 添加错误处理
- 优化性能

### 4. 测试验证
- 自动化测试
- 功能验证
- 性能测试
- 用户测试

---

## 触发条件

### 显式触发

- "生成技能"、"创建技能"、"新建技能"
- "需要XX技能"、"帮我做个XX技能"

- 技能发现后

## 技能类型

### 1. 数据处理类

**功能**: 处理和分析数据

**示例**:
- 数据清洗
- 数据转换
- 数据分析
- 数据可视化

**模板**:
```markdown
## 核心功能

### 1. 数据输入
- 支持格式: CSV, JSON, Excel
- 数据验证: 自动验证数据格式
- 错误处理: 提供错误提示

### 2. 数据处理
- 清洗: 去除重复、缺失值
- 转换: 格式转换、类型转换
- 分析: 统计分析、趋势分析

### 3. 数据输出
- 格式: CSV, JSON, Excel, Markdown
- 可视化: 图表、表格
- 报告: 分析报告
```

### 2. 文档生成类

**功能**: 自动生成文档

**示例**:
- 报告生成
- 文档编写
- 总结生成
- 摘要提取

**模板**:
```markdown
## 核心功能

### 1. 内容收集
- 来源: 文件、数据库、API
- 过滤: 关键词过滤
- 排序: 时间、相关性

### 2. 内容处理
- 整理: 分类、分组
- 分析: 关键词提取
- 总结: 自动总结

### 3. 文档生成
- 格式: Markdown, PDF, DOCX
- 结构: 章节目录
- 样式: 自定义样式
```

### 3. 自动化类

**功能**: 自动化重复性任务

**示例**:
- 文件处理
- 邮件发送
- 数据同步
- 定时任务

**模板**:
```markdown
## 核心功能

### 1. 任务调度
- 触发: 定时、事件驱动
- 优先级: 高、中、低
- 依赖: 任务依赖管理

### 2. 任务执行
- 执行: 自动执行
- 监控: 实时监控
- 重试: 失败重试

### 3. 结果处理
- 通知: 成功/失败通知
- 日志: 详细日志
- 统计: 执行统计
```

### 4. 分析类

**功能**: 数据分析和洞察

**示例**:
- 趋势分析
- 关联分析
- 预测分析
- 对比分析

**模板**:
```markdown
## 核心功能

### 1. 数据分析
- 统计: 描述性统计
- 趋势: 趋势分析
- 关联: 关联分析

### 2. 洞察发现
- 模式: 模式识别
- 异常: 异常检测
- 机会: 机会发现

### 3. 报告生成
- 图表: 可视化图表
- 报告: 分析报告
- 建议: 行动建议
```

## 代码生成规则

### 1. 文件命名

**格式**: `[技能名称]\SKILL.md`

**示例**:
- `数据处理\SKILL.md`
- `文档生成\SKILL.md`
- `自动化任务\SKILL.md`

### 2. 目录结构

```
.trae\skills\
├── [技能名称]\
│   ├── SKILL.md          # 技能定义
│   ├── templates\        # 模板文件
│   ├── examples\         # 示例文件
│   └── tests\            # 测试文件
```

### 3. 代码规范

- 使用 Markdown 格式
- 遵循现有技能结构
- 添加详细注释
- 提供使用示例
