# A/B 测试 Skill

## 概述

通过 A/B 测试验证优化方案的效果，对比不同版本的性能，为优化决策提供数据支持。

---

## 核心功能

### 1. 测试设计
- 定义测试目标
- 设计测试方案
- 确定测试指标
- 制定测试计划

### 2. 测试执行
- 创建测试版本
- 分配测试流量
- 执行测试
- 收集数据

### 3. 数据分析
- 统计测试数据
- 分析测试结果
- 计算显著性
- 评估效果

### 4. 结果报告
- 生成测试报告
- 对比测试结果
- 提供决策建议
- 规划后续行动

## 统计检验方法

### 1. t 检验

**用途**: 比较两组连续数据的均值

**公式**:
```
t = (X̄₁ - X̄₂) / √(s₁²/n₁ + s₂²/n₂)
```

**Python 实现**:
```python
from scipy import stats

def t_test(data_a, data_b):
    statistic, p_value = stats.ttest_ind(data_a, data_b)
    return {
        'statistic': statistic,
        'p_value': p_value,
        'significant': p_value < 0.05
    }
```

### 2. 卡方检验

**用途**: 比较两组分类数据的比例

**公式**:
```
χ² = Σ((O - E)² / E)
```

**Python 实现**:
```python
from scipy import stats

def chi2_test(observed_a, observed_b):
    observed = [observed_a, observed_b]
    chi2, p_value, dof, expected = stats.chi2_contingency(observed)
    return {
        'chi2': chi2,
        'p_value': p_value,
        'significant': p_value < 0.05
    }
```

### 3. Mann-Whitney U 检验

**用途**: 比较两组非正态分布数据

**Python 实现**:
```python
from scipy import stats

def mann_whitney_test(data_a, data_b):
    statistic, p_value = stats.mannwhitneyu(data_a, data_b)
    return {
        'statistic': statistic,
        'p_value': p_value,
        'significant': p_value < 0.05
    }
```

## 样本量计算

### 计算公式

```
n = (Zα/2 + Zβ)² × (σ₁² + σ₂²) / Δ²
```

其中:
- Zα/2: 显著性水平对应的 Z 值
- Zβ: 统计功效对应的 Z 值
- σ₁², σ₂²: 两组的方差
- Δ: 最小可检测效应

### Python 实现

```python
import math
from scipy import stats

def calculate_sample_size(effect_size, alpha=0.05, power=0.8):
    z_alpha = stats.norm.ppf(1 - alpha/2)
    z_beta = stats.norm.ppf(power)
    
    n = (z_alpha + z_beta)**2 * 2 / effect_size**2
    return math.ceil(n)
```
