# 任务处理 Skill

## 概述

确保每次接到任务都第一时间检查和使用相关的 skills，建立标准化的任务处理流程。

---

## 核心功能

### 1. 任务接收
- 接收用户任务
- 分析任务类型
- 识别任务关键词
- 确定任务优先级

### 2. 技能检查
- 检查现有 skills
- 匹配相关 skills
- 评估技能适用性
- 选择最佳技能

### 3. 技能使用
- 立即使用相关技能
- 执行任务处理
- 记录处理过程
- 生成处理结果

### 4. 技能发现
- 如果没有相关技能
- 使用技能发现 Skill
- 查找或创建新技能
- 更新技能库

---

## 触发条件

### 自动触发

- **所有任务**: 每次接到任务时自动触发
- **所有查询**: 每次接到查询时自动触发
- **所有请求**: 每次接到请求时自动触发

## 最佳实践

### 1. 第一时间检查技能

**原则**: 每次接到任务，第一时间检查相关 skills

**步骤**:
1. 接收任务
2. 分析任务关键词
3. 检查现有 skills
4. 匹配相关 skills
5. 立即使用技能

**示例**:
```
❌ 错误做法:
用户: 帮我整理文件夹
AI: 好的，让我先看看文件夹结构... (直接开始工作，没有检查 skills)

✅ 正确做法:
用户: 帮我整理文件夹
AI: 检测到"文件夹整理"任务，正在检查相关 skills...
AI: 找到"文件夹管理" Skill，立即使用...
```

### 2. 优先使用现有技能

**原则**: 优先使用现有的 skills，避免重复造轮子

**步骤**:
1. 检查现有 skills
2. 评估技能适用性
3. 选择最佳技能
4. 立即使用技能

**示例**:
```
❌ 错误做法:
用户: 帮我整理文件夹
AI: 好的，让我写一个脚本... (没有使用现有的技能)

✅ 正确做法:
用户: 帮我整理文件夹
AI: 检测到"文件夹整理"任务，正在检查相关 skills...
AI: 找到"文件夹管理" Skill，立即使用...
```

### 3. 没有技能时使用技能发现

**原则**: 如果没有相关技能，使用技能发现 Skill

**步骤**:
1. 检查现有 skills
2. 确认没有相关技能
3. 使用技能发现 Skill
4. 查找或创建新技能

**示例**:
```
❌ 错误做法:
用户: 我需要一个XXX技能
AI: 好的，让我创建一个... (没有使用技能发现)

✅ 正确做法:
用户: 我需要一个XXX技能
AI: 检测到新技能需求，正在使用"技能发现" Skill...
AI: 分析需求，推荐技能...
```

### 4. 记录任务处理过程

**原则**: 记录任务处理过程，便于后续优化

**步骤**:
1. 记录任务接收
2. 记录技能检查
3. 记录技能使用
4. 记录处理结果

**示例**:
```

- 技能检查: 找到"文件夹管理" Skill
```

## 自动化检查

### 任务处理检查脚本

```powershell
# 任务处理检查脚本
function Check-TaskProcessing {
    param (
        [string]$Task
    )

    # 1. 分析任务关键词
    $keywords = Get-TaskKeywords -Task $Task

    # 2. 检查相关 skills
    $skills = Get-RelevantSkills -Keywords $keywords

    # 3. 评估技能适用性
    $applicableSkills = Test-SkillApplicability -Skills $skills -Task $Task

    # 4. 选择最佳技能
    $bestSkill = Select-BestSkill -Skills $applicableSkills

    # 5. 返回结果
    return @{
        Task = $Task
        Keywords = $keywords
        RelevantSkills = $skills
        BestSkill = $bestSkill
    }
}

# 使用示例
$result = Check-TaskProcessing -Task "整理文件夹"
Write-Host "最佳技能: $($result.BestSkill)"
```

   - 增加任务类型识别
   - 改进适用性评估

   - 优化技能检查流程

   - 支持更多任务类型
