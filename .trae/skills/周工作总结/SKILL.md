# 周工作总结 Skill

## 基本信息

- **名称**: 周工作总结
- **功能**: 周工作总结生成工具 + 知识汇总
- **触发词**: "写周报"、"周工作总结"、"本周总结"
- **关键词**: Weekly report, work summary, 周报, 周工作总结, status report, progress report, work log, weekly update

---

## 功能说明

### 核心功能

1. **素材整合**: 收集一周的工作记录、日总结、新闻监测记录
2. **结构化输出**: 按照规定的5个必需内容生成周工作总结
3. **知识沉淀**: 提取可复用的知识、流程、工具
4. **知识汇总**: 汇总本周知识增量，更新知识库

### 知识汇总能力

| 汇总类型 | 汇总内容 | 更新位置 |
|---------|---------|---------|
| 项目进展汇总 | 本周各项目进展 | `项目库\项目跟踪记录.md` |
| 知识增量汇总 | 本周新增知识 | `个人知识库\学习记录.md` |
| 能力变化汇总 | 本周能力提升 | `个人知识库\能力图谱.md` |
| 关联发现汇总 | 本周发现的关联 | `项目库\_项目知识\项目关联图谱.md` |

---

## 输入输出

### 输入位置

| 输入类型 | 位置 |
|---------|------|
| 工作碎片 | `工作记录\`（一周的工作碎片）|
| 日总结 | `输出成果\日总结\`（一周的日总结）|
| 新闻记录 | `输出成果\新闻监测记录\`（一周的新闻）|
| 项目跟踪 | `项目库\项目跟踪记录.md`|
| 知识库 | `项目库\_项目知识\`、`个人知识库\`|

### 输出文件

| 输出类型 | 位置 | 文件命名 |
|---------|------|---------|
| 周工作总结 | `输出成果\周工作总结\` | `MM.DD-MM.DD周工作总结.md`|
| 归档 | `输出成果\归档\` | `2026年第X周\`|

## 周工作总结的5个必需内容

### 1. 总结、计划

```markdown
## 一、本周工作总结

### 1. 项目进展
- 项目A：...
- 项目B：...

### 2. 重点工作
- 工作项1：...
- 工作项2：...

## 二、下周工作计划

### 1. 重点工作
- [ ] 计划1
- [ ] 计划2

### 2. 待跟进事项
- 事项1
- 事项2
```

### 2. 新掌握的知识、常识列表

```markdown
## 三、新掌握的知识

### 知识点1：XXX
- **来源**: ...
- **内容**: ...
- **应用场景**: ...

### 知识点2：XXX
- **来源**: ...
- **内容**: ...
- **应用场景**: ...
```

```markdown
## 四、科普附件

### 新闻
1. [新闻标题](链接) - 简要说明
2. [新闻标题](链接) - 简要说明

```

```markdown

- 提示词1：用途说明

- 工具完善度：XX%

- 从项目库萃取了：XXX

- 完善了：XXX
```

## 工作流程

```
步骤1：收集一周素材
─────────────────────
工作记录\ （一周的工作碎片）
    ↓
输出成果\日总结\ （一周的日总结）
    ↓
输出成果\新闻监测记录\ （一周的新闻）
    ↓
项目库\项目跟踪记录.md
    ↓
项目库\_项目知识\ + 个人知识库\

步骤2：整合和分析
─────────────────────
所有素材 → 分类整理 → 提取关键信息

步骤3：知识汇总
─────────────────────
本周知识增量 → 汇总分析 → 更新知识库
     │
     ├── 项目进展 → 更新 项目跟踪记录.md
     ├── 知识增量 → 更新 学习记录.md
     ├── 能力变化 → 更新 能力图谱.md
     └── 关联发现 → 更新 项目关联图谱.md

步骤4：生成周工作总结
─────────────────────
关键信息 + 知识汇总结果 → 按5个必需内容组织 → 周工作总结
                                                    ↓
                            输出成果\周工作总结\MM.DD-MM.DD周工作总结.md

步骤5：用户确认和完善
─────────────────────
周总结 → 用户确认 → 补充完善

步骤6：归档
─────────────────────
周总结 + 日总结 + 新闻记录 → 输出成果\归档\2026年第X周\
```
